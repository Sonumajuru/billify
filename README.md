# Billify — Professional Invoices & Tenancy Receipts

> Professional invoices and tenancy receipts, built for the Netherlands — in seconds.

## What is Billify?
Billify is a document generation tool for IT professionals and landlords. Create polished invoices, quotes, and tenancy receipts using six designer templates. Built with Dutch compliance in mind — VAT (21%/9%/0%), KvK, IBAN, BTW reverse charge, and iDEAL all included. Save your work, reuse your branding, and export to PDF in one click.

## Features
- **IT Invoice / Quote** — 6 templates, Dutch VAT, KvK, IBAN, reverse charge, EU intra-community
- **Tenancy Receipt** — 6 templates, rent + utilities + deposit breakdown
- **Mode switcher** — switch between Invoice and Tenancy in one click
- **Save & Resume** — save documents to localStorage, reload any time
- **Templates** — save your branding/design as a reusable template
- **Export/Import templates** — download as .json, share or backup
- **Auto-save draft** — saves every 3 seconds while editing
- **PDF download** — high-resolution export
- **Print** — full browser print support
- **No account required** — all data stays in your browser
